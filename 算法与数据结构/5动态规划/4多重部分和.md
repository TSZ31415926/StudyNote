# 1
1. 准备
```C++
int n,k;
vector<int> val(n+1),m(n+1),dp(k+1,-1);
```

2. 核心代码
```C++
for(int i=1;i<=n;i++){
    for(int j=1;j<=k;j++){
        if(dp[j]>=0){//情况1
            dp[j]=m[i];//若已能被表示，则对j来说保存实力，一个不用
        }
        else if(j>=val[i]&&dp[j-val[i]]){//情况2
            dp[j]=dp[j-val[i]]-1;
        }
        else{//情况3
            dp[j]=-1;
        }
    }
}
cout<<(dp[k]>=0?"Yes":"No")<<'\n';
```
- 对于**情况1**：若已能被表示，则**对j来说**保存实力，一个不用
- 对于**情况2**：无法直接组成j，但**若可以**通过消耗当前物品组成，则使用一个当前物品i，剩余数量减1
- **情况3**：无法组成当前金额j
